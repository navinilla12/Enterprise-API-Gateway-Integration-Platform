# Enterprise API Gateway & Integration Platform


## Overview

A production-style API integration platform designed to connect multiple SaaS applications through secure REST APIs.

The platform demonstrates enterprise backend engineering practices including authentication, authorization, API security, database integration, containerization, and cloud deployment.


## Tech Stack

- Python
- FastAPI
- PostgreSQL
- AWS API Gateway
- AWS EC2
- AWS RDS
- Docker
- OAuth2
- JWT
- OpenAPI Swagger
- GitHub Actions


## Features

### API Gateway

- REST API architecture
- OpenAPI documentation
- SaaS integration endpoints


### Security

- OAuth2 authentication
- JWT token management
- API key validation
- Role-based access control


### Cloud Architecture

- Dockerized deployment
- AWS EC2 hosting
- PostgreSQL RDS database
- CI/CD automation


## Analytics

Includes API monitoring:

- Request volume analysis
- Latency monitoring
- Error-rate tracking
- SLA health scoring


## Run Locally


Clone repository:
git clone repo-url


Install dependencies:
pip install -r requirements.txt


Start server:
uvicorn main:app --reload


Swagger Documentation:
localhost:8000/docs
